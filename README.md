# Sales Data Analysis

**Author:** Manbendra Singh  
**Role Target:** Data Analyst

## Project Overview
This project focuses on analyzing sales data to identify trends, top-performing products, and regional performance. The analysis helps businesses make data-driven decisions and reduce manual reporting efforts.

## Tools Used
- Excel / Google Sheets
- SQL (MySQL)
- Data Visualization

## Key Analysis Performed
- Monthly sales trend analysis
- Top 5 products by revenue
- Region-wise sales performance
- Profit and revenue comparison

## Key Insights
- Identified high-performing products contributing maximum revenue
- Observed seasonal sales patterns
- Highlighted underperforming regions for improvement

## Outcome
Improved understanding of sales trends and supported business decision-making through data insights.
